# Sequential Pattern Generator

このサイトは数列的に文字・図形・画像をパターン配置するものとなっています。
各項目の説明文はサイト右上の「？」マークを押すと読めます。

※このサイトはClaudeを用いて制作しています。また、当方はプログラミングについて全く明るくなく、各処理が厳密ではない可能性があります。ご了承ください。

--------------------------------------
This site arranges text, shapes, and images in patterns based on numerical sequences.
You can view descriptions for each item by clicking the “?” icon in the upper-right corner of the site.

*This site was created using Claude. Please note that, as I have no programming experience, the code may not be entirely accurate. Thank you for your understanding.
